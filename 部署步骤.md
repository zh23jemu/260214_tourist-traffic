# 部署步骤

## 1. 获取项目

```powershell
git clone <你的仓库地址>
cd 260214_tourist-traffic
```

如果项目已经在本机，直接进入项目根目录即可。

## 2. 复制环境变量文件

```powershell
Copy-Item .env.example .env
```

## 3. 准备数据库（Windows 本机 MySQL 8）

启动 MySQL 服务：

```powershell
Start-Service MySQL80
```

登录 MySQL：

```powershell
mysql -u root -p
```

创建数据库和账号：

```sql
CREATE DATABASE tourist_traffic CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'tourist'@'localhost' IDENTIFIED BY 'tourist';
GRANT ALL PRIVILEGES ON tourist_traffic.* TO 'tourist'@'localhost';
FLUSH PRIVILEGES;
```

## 4. 创建后端虚拟环境

```powershell
py -3.12 -m venv venv
.\venv\Scripts\Activate.ps1
```

## 5. 安装 Python 依赖

```powershell
.\venv\Scripts\python.exe -m pip install --upgrade pip
.\venv\Scripts\python.exe -m pip install -r requirements.txt
```

## 6. 启动后端

```powershell
.\venv\Scripts\python.exe -m uvicorn app.main:app --reload
```

后端地址：

```text
http://127.0.0.1:8000
```

接口文档：

```text
http://127.0.0.1:8000/docs
```

## 7. 启动前端

```powershell
cd frontend
npm install
npm run dev
```

前端地址通常为：

```text
http://127.0.0.1:3000
```

## 8. 启动研究系统

如果要运行研究系统，在项目根目录执行：

```powershell
py -3.12 -m venv venv312
.\venv312\Scripts\python.exe -m pip install --upgrade pip
.\venv312\Scripts\python.exe -m pip install -r requirements.txt
.\venv312\Scripts\python.exe -m streamlit run research_app\app.py --server.address 127.0.0.1 --server.port 18080
```

研究系统地址：

```text
http://127.0.0.1:18080
```

## 9. 登录系统

默认账号：

```text
用户名：admin
密码：admin123
```

## 10. 初始化数据

登录前端后，按这个顺序操作：

1. 数据导入
2. 特征构建
3. 模型训练
4. 生成预测

导入文件优先使用：

```text
毕设数据统计.csv
```

## 11. 常用启动顺序

1. 启动 MySQL
2. 启动后端
3. 启动前端
4. 如需论文实验，再启动研究系统
